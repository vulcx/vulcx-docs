llmDescription: "OpenAPI reference for POST /api/v1/cpi/route-accounts on the Vulcx API (Fogo). Builds the `route` instruction shaped for a Cross-Program Invocation: the request supplies a calling program's PDA as the swap authority (signer, writable when a Moonit hop is involved) instead of a wallet. Response returns a single RouteInstruction (RawInstruction with accounts/data) to forward into the program's route CPI and sign with invoke_signed, plus requiredTokenAccounts (ATAs the PDA must own and fund — no ATA-create or wrap/unwrap instructions are emitted) and addressLookupTableAddresses for a v0 transaction. Split routes are not yet supported (422). Beta. Interactive playground included."
